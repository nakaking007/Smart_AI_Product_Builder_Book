# Smart AI Product Builder — Complete Edition

A professional two-day Complete Edition for turning AI ideas into usable digital products. Future topic-specific books remain an Extended Series plan and are not currently in production.

## Project status

The publisher-edition production is active for one canonical book that combines Basic, Advanced, Single Web App, Workshop, Prompt Library, appendices, and Capstone content in one complete PDF.

See:

- `PROJECT_STATE.md` for the current working status.
- `BOOK_SPEC.md` for the book's purpose, audience, scope, and deliverables.
- `CONTENT_PLAN.md` for the chapter-level content map.
- `STYLE_GUIDE.md` for writing and formatting rules.
- `AGENTS.md` for contributor and AI-agent instructions.

## Directory guide

- `chapters/` — manuscript source files, one Markdown file per chapter.
- `assets/` — cover art, images, icons, diagrams, and QR codes.
- `prompts/` — reusable prompts referenced by the book.
- `code/` — complete code samples and example projects.
- `templates/` — reusable reader worksheets and project templates.
- `references/` — research notes, citations, and source records.
- `output/` — generated publication files; not the manuscript source.
- `scripts/` — build, validation, and publishing utilities.
- `BOOK_SET_01/` — cover, preface, reader guide, contents, and the Gem chapter.
- `SERIES/` — Complete Edition production source plus paused future-series plans.

## Editorial workflow

1. Confirm the book specification and content plan.
2. Research and record authoritative sources.
3. Draft one chapter at a time.
4. Validate every prompt, procedure, and code example.
5. Perform technical and editorial reviews.
6. Generate and inspect publication outputs.
