# Smart AI Product Builder Book

A professional, practical book project for turning AI ideas into usable digital products—starting with AI foundations and prompt engineering, then progressing through Gems, Custom GPTs, NotebookLM, single-file HTML apps, Google Apps Script, Chrome extensions, Codex, and complete AI product development.

## Project status

See:

## Directory guide

- `chapters/` — manuscript source files, one Markdown file per chapter.
- `assets/` — cover art, images, icons, diagrams, and QR codes.
- `prompts/` — reusable prompts referenced by the book.
- `code/` — complete code samples and example projects.
- `templates/` — reusable reader worksheets and project templates.
- `references/` — research notes, citations, and source records.
- `output/` — generated publication files; not the manuscript source.
- `scripts/` — build, validation, and publishing utilities.

## Editorial workflow

1. Confirm the book specification and content plan.
2. Research and record authoritative sources.
3. Draft one chapter at a time.
4. Validate every prompt, procedure, and code example.
5. Perform technical and editorial reviews.
6. Generate and inspect publication outputs.
