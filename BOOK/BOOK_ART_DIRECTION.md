# Book Art Direction

## Creative Direction

- Theme: Modern Education
- Character: Professional, Friendly, Practical
- Style: Minimal Flat Design with restrained glass-like accents
- Primary colors: Blue, Green and Gold
- หลีกเลี่ยง Emoji ปะปนในภาพวิชาการ ใช้ Icon System เดียวกัน

## Visual Rhythm

- วางภาพหรือองค์ประกอบภาพอย่างน้อยหนึ่งรายการต่อประมาณ 2 หน้าจัดพิมพ์
- ทุกบทมี Hero Banner, Screenshot, Diagram, Infographic และ Summary Visual
- Full-page Infographic ใช้เมื่อเนื้อหามี Framework หรือ Architecture ซับซ้อน
- ภาพต้องช่วยสอน ห้ามใช้เพียงตกแต่ง

## Numbering

- Screenshot, Illustration และ Infographic ใช้เลขภาพตามบท เช่น ภาพที่ 3-2
- Diagram ใช้เลขแผนภาพตามบท
- Table ใช้เลขตารางตามบท
- Caption ต้องอธิบายสิ่งที่ผู้อ่านควรสังเกต

## Typography in Artwork

- Heading: Sarabun Bold
- Body: Sarabun
- Code: JetBrains Mono
- ห้ามใช้ตัวอักษรเล็กจนอ่านไม่ได้เมื่อพิมพ์ A5

## Artwork Package

ทุกภาพต้องมี

- Source File
- Export File
- Image Prompt หากสร้างด้วย AI
- Aspect Ratio
- Color Palette
- Alt Text
- Copyright หรือ License Status

## Print

- Raster target: 300 PPI ที่ขนาดใช้งานจริง
- ใช้ Vector สำหรับ Diagram และ Icon เมื่อทำได้
- ปกหรือภาพตกขอบต้องเผื่อ Bleed ตามโรงพิมพ์
- ตรวจ Proof บนกระดาษก่อนอนุมัติผลิต
