# วิธีใช้หนังสือ

หนังสือเล่มนี้ออกแบบให้เปิดเครื่องแล้วทำตามได้ทันที ผู้อ่านไม่จำเป็นต้องมีพื้นฐานการเขียนโปรแกรม แต่ควรทำแต่ละ Step ตามลำดับและตรวจผลงานก่อนข้ามไปบทถัดไป

## เส้นทางเรียน 2 วัน

| วัน | หัวข้อ | ผลงานปลายวัน |
| --- | --- | --- |
| วันที่ 1: Basic | Gem, Custom GPT, วิเคราะห์ตลาดและราคา, NotebookLM | ผู้ช่วย AI และคลังความรู้ต้นแบบ |
| วันที่ 2: Advanced | Chrome Extension, Google Apps Script, Codex, การเผยแพร่ AI Product | AI Product ต้นแบบพร้อมแผนเผยแพร่ |

## วิธีเรียนในแต่ละบท

1. อ่าน **Pain Point** เพื่อเข้าใจปัญหาที่กำลังแก้
2. ตรวจ **สิ่งที่ต้องเตรียม** และสำรองไฟล์ก่อนเริ่ม
3. ทำ **Step by Step** ตามลำดับ
4. ใช้ **Prompt พร้อมใช้** แล้วแทนข้อความในวงเล็บเหลี่ยมด้วยข้อมูลของตนเอง
5. ทำ **Workshop** เพื่อฝึกตามตัวอย่าง
6. ทำ **Mini Project** เพื่อสร้างผลงานของตนเอง
7. ตรวจ **Checklist** และตอบแบบฝึกหัดก่อนขึ้นบทใหม่

## สัญลักษณ์ที่ใช้

| สัญลักษณ์ | ความหมาย |
| --- | --- |
| **AI Tip** | เทคนิคที่ช่วยลดเวลาและเพิ่มคุณภาพ |
| **ระวัง** | จุดที่อาจทำให้ข้อมูลเสียหาย รั่วไหล หรือได้ผลลัพธ์ผิด |
| **ตรวจผลงาน** | เกณฑ์ที่ผู้อ่านใช้ยืนยันว่าทำสำเร็จ |
| `[SCREENSHOT-xx-xx]` | ตำแหน่งภาพหน้าจอที่จะจัดทำในขั้น Artwork |
| `[QR-xx]` | ตำแหน่ง QR Code ที่รอ URL ปลายทางจริง |

## ก่อนเริ่มเรียน

- ใช้คอมพิวเตอร์ที่เชื่อมต่ออินเทอร์เน็ตและมีเว็บเบราว์เซอร์รุ่นปัจจุบัน
- เตรียมบัญชี Google และบัญชีบริการ AI ตามบทที่เรียน
- สร้างโฟลเดอร์เก็บ Prompt, ไฟล์ตัวอย่าง และผลงานแยกเป็นรายบท
- ใช้ข้อมูลตัวอย่างในระหว่างฝึก ห้ามอัปโหลดรหัสผ่าน ข้อมูลสุขภาพ ข้อมูลนักเรียน หรือข้อมูลลับขององค์กร
- หน้าจอและความสามารถของบริการอาจเปลี่ยนได้ ให้ตรวจวันที่กำกับใน Menu Map ของแต่ละบท

## วิธีใช้ Prompt Box

1. อ่านวัตถุประสงค์และเลือกเครื่องมือให้ตรงกับที่ระบุ
2. คัดลอก Prompt ทั้งชุด
3. แทนค่าข้อความใน `[วงเล็บเหลี่ยม]`
4. ตรวจข้อมูลส่วนบุคคลก่อนส่ง
5. เปรียบเทียบผลลัพธ์กับเกณฑ์ตรวจใน Prompt Box
6. บันทึก Prompt ที่ผ่านการทดสอบเป็นเวอร์ชันของตนเอง
