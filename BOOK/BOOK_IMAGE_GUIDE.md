# Book Image Guide

## Theme

ใช้ภาพแนว Modern Education แบบเรียบ สะอาด และเป็นวิชาการ

## Color Palette

- Primary: #1E88E5
- Secondary: #00ACC1
- Success: #43A047
- Warning: #FB8C00
- Danger: #E53935
- Background: #FAFAFA
- Ink: #17365D

## ประเภทภาพ

- Screenshot
- Diagram
- Workflow
- Infographic
- Icon
- Timeline
- Mind Map
- Architecture Diagram

## กติกา Placeholder

Placeholder ทุกภาพต้องระบุ

1. หมายเลขภาพ
2. ชื่อภาพ
3. วัตถุประสงค์
4. องค์ประกอบที่ต้องเห็น
5. Callout หรือลูกศร
6. Aspect Ratio
7. Alt Text
8. แหล่งที่มาหรือสถานะลิขสิทธิ์

## Screenshot

- หนึ่งขั้นตอนสำคัญต่อหนึ่งภาพ
- ปิดบังชื่อ อีเมล Token และข้อมูลส่วนบุคคล
- ระบุวันที่ตรวจสอบ UI
- ไม่ใช้ภาพหน้าจอที่อ่านตัวอักษรไม่ได้
- ใช้กรอบและหมายเลข Callout แบบเดียวกันทั้งเล่ม

## Infographic

- ใช้เมื่อช่วยอธิบาย Framework, Workflow, Lifecycle, Matrix หรือ Architecture
- หนึ่งภาพสื่อสารประเด็นหลักเพียงเรื่องเดียว
- ต้องมี Text Alternative ในเนื้อหา

## QR Code

- สร้างเมื่อมี URL ปลายทางจริงและผ่านการทดสอบแล้ว
- ใต้ QR Code ต้องมีชื่อปลายทางและ URL แบบอ่านได้
- เว้น Quiet Zone รอบ QR Code
- ทดสอบจากไฟล์ PDF และกระดาษพิมพ์จริง

## คุณภาพไฟล์

- ภาพพิมพ์เป้าหมาย 300 PPI ที่ขนาดใช้งานจริง
- Diagram และ Icon ใช้ SVG หรือ Vector เมื่อระบบส่งออกสนับสนุน
- ใช้ PNG สำหรับ Screenshot
- CMYK, Bleed และ Crop Mark ใช้ตามข้อกำหนดโรงพิมพ์ โดยเฉพาะปกหรือหน้าที่พิมพ์ตกขอบ
