# Book Style Guide

## ผู้อ่านและน้ำเสียง

- เขียนภาษาไทยอ่านง่าย เหมือนผู้สอนยืนอยู่ข้างผู้เรียน
- สมมติว่าผู้อ่านไม่มีพื้นฐาน Coding
- อธิบายเหตุผลสั้น ๆ ก่อนขั้นตอน
- ใช้ประโยคกระชับและคำกริยาที่ลงมือทำได้
- ไม่ใช้คำโฆษณาเกินจริง เช่น ง่ายที่สุด สมบูรณ์แบบ หรือรับประกัน
- หากทำไม่ได้หรือมีข้อจำกัด ให้บอกตรงไปตรงมา

## Menu Map ของเครื่องมือ

- บทที่สอนเครื่องมือต้องอธิบายเมนูที่เกี่ยวข้องครบตามขอบเขตบท
- เริ่มด้วยตาราง Menu Map: ชื่อเมนู หน้าที่ ใช้เมื่อใด และข้อควรระวัง
- ระบุวันที่ตรวจสอบหน้าจอ เพราะชื่อและตำแหน่งเมนูอาจเปลี่ยนได้
- หากยังไม่ได้ตรวจหน้าจอจริง ให้ติดป้าย **รอตรวจสอบ UI** ห้ามเขียนเหมือนเป็นข้อมูลยืนยัน
- ไม่ต้องอธิบายเมนูผู้ดูแลระบบหรือเมนูนอกขอบเขตโดยไม่มีเหตุผล แต่ต้องบอกผู้อ่านว่าตัดออกเพราะอะไร

## กติกา Markdown

- ใช้ H1 หนึ่งครั้งสำหรับชื่อบท
- ใช้ H2 สำหรับหัวข้อหลัก และ H3 สำหรับหัวข้อย่อย
- ไม่ข้ามระดับหัวข้อ
- ใช้รายการตัวเลขเฉพาะขั้นตอนที่ต้องเรียงลำดับ
- ใช้ Bullet สำหรับตัวเลือกหรือ Checklist
- Code และ Prompt ใช้ fenced code block พร้อมชื่อภาษา
- ตารางใช้เฉพาะข้อมูลที่ต้องเปรียบเทียบ
- ลิงก์ใช้ข้อความอธิบาย ห้ามวาง URL เปล่า

## Prompt Box

Prompt ทุกชุดต้องระบุ

- ชื่อ Prompt
- เครื่องมือที่เหมาะสม
- ระดับผู้ใช้
- วัตถุประสงค์
- Prompt ฉบับเต็ม
- ผลลัพธ์ที่คาดหวัง
- วิธีตรวจสอบ
- ข้อควรระวัง

## Placeholder สำหรับงานศิลป์

- Screenshot Placeholder ระบุเลขภาพ หน้าจอ จุด Callout และ Alt Text
- Infographic Placeholder ระบุหัวข้อ ข้อมูลที่จะนำเสนอ และลำดับการอ่าน
- QR Code Placeholder ระบุปลายทาง วัตถุประสงค์ และข้อความสำรองกรณีสแกนไม่ได้
- ห้ามสร้าง QR Code ที่ยังไม่มี URL ปลายทางจริง
- Placeholder ต้องอยู่ในตำแหน่งที่จะใช้จริงเมื่อนำภาพมาแทน เพื่อไม่ต้องจัดโครงสร้างใหม่ก่อน Export

## Code Box

Code ทุกชุดต้องระบุ

- ชื่อไฟล์
- ภาษา
- หน้าที่
- Code ฉบับเต็ม
- วิธีรัน
- ผลลัพธ์ที่คาดหวัง
- วิธีทดสอบ

## ความถูกต้อง

- ใช้ข้อมูลจริงหรือระบุว่าเป็นข้อมูลตัวอย่าง
- ไม่สร้างแหล่งอ้างอิง กฎหมาย สถิติ หรือผลทดสอบขึ้นเอง
- ความสามารถของผลิตภัณฑ์ที่เปลี่ยนแปลงได้ต้องตรวจสอบก่อนเผยแพร่
- งานสุขภาพ กฎหมาย และการเงินต้องมีคำเตือนและผู้เชี่ยวชาญตรวจทาน
- ห้ามใส่รหัสผ่าน Token หรือข้อมูลส่วนบุคคลจริง
