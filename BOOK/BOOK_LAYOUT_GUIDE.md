# Book Layout Guide

## รูปเล่มมาตรฐาน

- ขนาดสำเร็จ: A5 (148 × 210 มม.)
- พิมพ์สองหน้าแบบหน้าคี่–หน้าคู่
- ขอบใน: 22 มม.
- ขอบนอก: 15 มม.
- Gutter: 3 มม.
- ขอบบน: 18 มม.
- ขอบล่าง: 20 มม.
- บทใหม่เริ่มหน้าคี่
- หน้าแรกของบทไม่มี Running Header

## Typography

- ภาษาไทย: TH Sarabun New, Sarabun หรือฟอนต์ไทยที่มี License เหมาะกับการพิมพ์
- เนื้อหา: 14–16 pt ตามฟอนต์
- ระยะบรรทัด: 1.05–1.20
- หัวบท: 28–32 pt
- หัวข้อหลัก: 21–24 pt
- หัวข้อย่อย: 16–18 pt
- Code: JetBrains Mono หรือ Consolas 8.5–10 pt

## ระบบหน้า

- หน้าคู่: ชื่อหนังสือที่หัวกระดาษด้านนอก
- หน้าคี่: ชื่อบทที่หัวกระดาษด้านนอก
- เลขหน้าอยู่มุมล่างด้านนอก
- ใช้ Mirror Margins และเผื่อสันเข้าเล่ม
- ป้องกัน Widow, Orphan และหัวข้อโดดท้ายหน้า
- หัวตารางต้องซ้ำเมื่อข้ามหน้า

## กล่องเนื้อหา

ใช้รูปแบบคงที่สำหรับ Tip, Warning, Note, Goal, Concept, Workshop, Prompt และ Case Study โดยใช้ข้อความกำกับร่วมกับสี ห้ามสื่อความหมายด้วยสีเพียงอย่างเดียว

## ความพร้อมสำหรับ PDF

- Placeholder ทุกชนิดอยู่ในตำแหน่งจริงตั้งแต่ต้น
- รูปต้องมี Caption และ Alt Text
- ตารางต้องมีความกว้างแน่นอนและไม่ล้นขอบ
- Code block ต้องตัดบรรทัดหรือแบ่งหน้าได้
- ใช้ Page Break และ Section Break ตามบท ห้ามใช้บรรทัดว่างดันหน้า
