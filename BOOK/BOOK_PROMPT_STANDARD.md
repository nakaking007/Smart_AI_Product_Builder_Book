# Book Prompt Standard

## Metadata บังคับ

Prompt ทุกชุดต้องมี

- Prompt Name
- AI Tool
- Difficulty
- Use Case
- Objective
- Prompt
- Expected Output
- Verification
- Tips
- Limitations

## รูปแบบ

~~~markdown
### [Prompt Name]

- **AI Tool:** [เครื่องมือ]
- **Difficulty:** Beginner / Intermediate / Advanced
- **Use Case:** [สถานการณ์]
- **Objective:** [ผลลัพธ์]

~~~text
[Prompt ฉบับเต็ม]
~~~

**Expected Output:** [คำอธิบาย]

**Verification:** [เกณฑ์ตรวจ]

**Tips:** [วิธีปรับใช้]

**Limitations:** [ข้อจำกัด]
~~~

## กติกาคุณภาพ

- Prompt ต้องนำไปทดลองได้โดยไม่เติมข้อมูลสำคัญที่ขาด
- ใช้ Placeholder ที่สื่อความหมาย
- มีวิธีจัดการเมื่อข้อมูลไม่ครบ
- ห้ามขอให้ AI สร้างแหล่งอ้างอิงหรือข้อมูลที่ไม่มีหลักฐาน
- ระบุสิ่งที่มนุษย์ต้องตรวจ
- Prompt ที่ผ่านการทดสอบต้องมี Version และ Test Case
