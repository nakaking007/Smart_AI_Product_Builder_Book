# Smart AI Product Builder — Series Blueprint

## วิสัยทัศน์

สร้างชุดหนังสือภาคปฏิบัติที่ผู้อ่านหยิบเรียนเป็นรายทักษะได้ อัปเดตแยกเล่มได้ และรวมเป็น Box Set ที่มีภาพลักษณ์เดียวกัน หนังสือทุกเล่มต้องเป็น How To อย่างน้อย 80% มีผลงานส่งมอบ และไม่อ้างฟีเจอร์ที่ไม่ได้ตรวจจากเอกสารทางการ

## ระบบภาพปกแบบ Series

- ใช้กริด โลโก้ ชื่อชุด ตำแหน่งผู้เขียน และ Typography เดียวกัน

- ใช้ Navy Blue เป็นสีหลัก Black/White เป็นฐาน และ Gold เป็น Accent ร่วม

## Definition of Done ต่อบท

- หน้าเปิดบทเต็มหน้าและ Opening Quote
- Pain Point, Learning Objectives และ Prerequisite
- Menu Map หรือ Concept Map ที่ตรวจสอบวันที่แล้ว
- Step by Step พร้อม Screenshot Placeholder ทุกขั้น
- Architecture, Workflow, Mind Map และ Infographic อย่างละ 1 จุดเมื่อสัมพันธ์กับเนื้อหา
- Prompt Box พร้อมเกณฑ์ตรวจผล
- Case Study, Workshop, Mini Project และ Checklist
- Quote อย่างน้อย 2 จุด
- QR Placeholder ที่ไม่สร้าง QR ปลอม
- Official References และวันที่ Fact Check
- Final Art Director Check ผ่าน

## Release Strategy

1. Alpha: โครงสร้างและเนื้อหาครบ ตรวจ Content Parity
2. Beta: Fact Check, Technical Review และ Workshop Trial
3. Release Candidate: DOCX/PDF/EPUB พร้อม Render QA
4. v1.0: ไฟล์โรงพิมพ์ ภาพ CMYK 300 DPI และ Errata Channel
5. Box Set: ตรวจสันปก ลำดับสี Metadata และ Cross-promotion ทั้งชุด

## Single-PDF Book Rule

- หนึ่งเล่มส่งมอบเป็น PDF หลักหนึ่งไฟล์
- Front Matter, เนื้อหา 20 บท, ภาคผนวก, References และ Index ต้องรวมอยู่ในไฟล์เดียว
- ใช้ Bookmark และสารบัญแบบคลิกได้เพื่อไปยังบท โดยไม่แยกไฟล์
- PDF ต้องฝังฟอนต์ มีเลขหน้าต่อเนื่อง มี Bleed/Crop ตามฉบับโรงพิมพ์ และผ่าน Preflight
- ไฟล์ Proof และไฟล์ภาพ Render เป็น QA Artifact ไม่ถือเป็น PDF ฉบับจำหน่าย
- เมื่อออก Revision ให้แทนด้วยไฟล์เวอร์ชันใหม่ทั้งเล่ม ไม่แจก Patch PDF รายบท
