# Thai Visual Guide — Complete Edition

## เป้าหมาย

สร้างภาพประกอบที่ผู้อ่านไทยรู้สึกใกล้ตัว เห็นตนเองและบริบทการทำงานจริง โดยยังคงคุณภาพ Editorial Illustration ระดับหนังสือคอมพิวเตอร์ 4 สี

## กลุ่มบุคคลที่ควรปรากฏ

| กลุ่ม | บริบทภาพ |
| --- | --- |
| นักเรียนและนักศึกษาไทย | ห้องเรียน ห้องสมุด ห้องปฏิบัติการ และการทำโครงงาน |
| ครูและอาจารย์ไทย | วางแผนการสอน ตรวจงาน และสร้างสื่อด้วย AI |
| บุคลากรภาครัฐ/อปท. | สำนักงานประชาชน งานเอกสาร และ Dashboard โดยไม่ใช้ตราปลอม |
| ผู้ประกอบการ SME ไทย | ร้านค้าออนไลน์ ร้านอาหาร งานบริการ และการตลาด |
| นักพัฒนาไทย | Laptop, Code Editor, Repository และการทดสอบระบบ |
| ผู้บริหารและทีมงาน | ประชุม วิเคราะห์ข้อมูล และตัดสินใจจาก Dashboard |
| ผู้สูงอายุและผู้พิการ | การใช้เทคโนโลยีที่เข้าถึงได้ในบริบทที่เหมาะสม |

## Style

- ใช้ Editorial Illustration แบบกึ่งสมจริงหรือ Flat Vector ที่มีเส้นและแสงสม่ำเสมอ
- สีหลัก Navy, Green, Orange และ Gold ตามระบบหนังสือ
- ฉากสะอาด มี White Space สำหรับ Caption หรือ Callout
- อัตราส่วนกำหนดตาม Page Type: Hero 4:3/16:9, Inline 3:2, Portrait 4:5, Icon 1:1
- หลีกเลี่ยงรายละเอียดจอภาพที่เป็นข้อความมั่ว หากต้องอ่านให้สร้าง UI แยกเป็น Screenshot/Mockup

## Authenticity Rules

- บุคลิกและใบหน้าเป็นคนไทยอย่างเป็นธรรมชาติ ไม่ทำให้ทุกคนมีหน้าตาเหมือนกัน
- ใช้สภาพแวดล้อมและอุปกรณ์ร่วมสมัย ไม่ตกแต่งความเป็นไทยเกินจริง
- เครื่องแบบราชการ ชุดนักเรียน และตราสัญลักษณ์ใช้เฉพาะเมื่อมีต้นฉบับและสิทธิ์ที่ตรวจสอบแล้ว
- เมื่อไม่แน่ใจ ให้ใช้ชุดทำงานทั่วไปและฉากสำนักงานไทยแบบไม่ระบุหน่วยงาน
- ตรวจมือ นิ้ว ตา ฟัน ตัวอักษรไทย หน้าจอ อุปกรณ์ และเงาสะท้อนทุกภาพ

กิจกรรม: [สิ่งที่กำลังทำ]

พื้นที่ว่าง: เว้นพื้นที่สำหรับ [หัวข้อ/Callout/Caption]

Lighting: soft professional daylight

garbled Thai text, duplicated people, excessive neon, heavy shadow, cheap clip-art style

## Asset Record

ทุกภาพต้องบันทึก Asset ID, ชื่อไฟล์, บท/หน้าเป้าหมาย, Prompt, Tool/Model, วันที่สร้าง, ผู้ตรวจ, Alt Text, Caption, Effective PPI, Color Profile, Source/Rights และ Approval Status

## Approval Checklist

- [ ] บุคคลและฉากเป็นบริบทไทยที่เหมาะสม
- [ ] ไม่สร้างภาพเหมารวมหรือใช้สัญลักษณ์ไทยเกินจำเป็น
- [ ] ไม่มีโลโก้ ตรา หรือบุคคลจริงที่ไม่มีสิทธิ์
- [ ] มือ ใบหน้า ตัวอักษร และอุปกรณ์ถูกต้อง
- [ ] สไตล์และ Palette ตรงทั้งเล่ม
- [ ] ภาพมีพื้นที่สำหรับเนื้อหาและไม่แน่นเกินไป
- [ ] Alt Text, Caption และ Asset Metadata ครบ
- [ ] 300 PPI/Vector และ CMYK พร้อมสำหรับฉบับพิมพ์
