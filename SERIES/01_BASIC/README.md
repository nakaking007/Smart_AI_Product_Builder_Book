# Smart AI Product Builder — Complete Edition

**ชื่อรอง:** คู่มือเรียนการใช้ AI ฉบับจับมือทำ

**รหัสเล่ม:** SAPB-COMP
**สถานะ:** Premium Edition — In Production

หนังสือหลักสูตร 2 วันฉบับสมบูรณ์ รวม Gem, Custom GPT, NotebookLM, Single Web App, Chrome Extension, Google Apps Script, Codex, AI Business, Workshop และ Capstone ไว้ในเล่มเดียว

## เป้าหมายผู้อ่าน

- เริ่มใช้ AI อย่างมีโครงสร้าง
- สร้างผู้ช่วย AI ที่ทำงานซ้ำได้
- ทดสอบและแชร์อย่างปลอดภัย
- อธิบายคุณค่าและตั้งราคาเบื้องต้นได้
- สร้าง Single File Web App ที่เปิดใช้งานได้จริง
- จบหลักสูตรด้วย Capstone AI Product และแฟ้มหลักฐาน

ใช้ `BOOK_PREMIUM_DECORATION_STANDARD.md` และ `SERIES_ART_DIRECTION.md` เป็น Design Authority
