### รายการแก้เฉพาะ Concept v0.2

- ตัดคำว่า `BEST SELLER` จนกว่าจะมีหลักฐานยอดขายที่ตรวจสอบได้
- ตัดหรือเปลี่ยน `ขายได้` และ `หารายได้จริง` เป็น “พร้อมต่อยอดสู่การใช้งานและธุรกิจ”
- ตัดป้าย `LMS` หากไม่ใช่ชื่อรุ่นหรือขอบเขตที่อนุมัติ
- ตัวเลข `30+ AI Products`, `300+ Prompt`, `200+ Code` ใช้ได้เมื่อรายงานฉบับสุดท้ายยืนยันจำนวนจริงเท่านั้น
- โลโก้ ChatGPT, Gemini, NotebookLM, Google Apps Script และ Chrome ต้องใช้ตาม Brand Guidelines และสิทธิ์ที่เกี่ยวข้อง
- เปลี่ยนโค้ดและหน้าจอที่อ่านได้ยาก/ล้าสมัยเป็น UI และ Code Example ที่ตรวจจากบทจริง
- แก้ข้อความจำลองบนโทรศัพท์ซึ่งอ่านไม่ถูกต้อง
- เปลี่ยนผู้จัดทำด้านล่างให้ตรง Metadata ผู้เขียน/กองบรรณาธิการที่อนุมัติ

## ผู้เขียนและราคา

- **ข้อความผู้เขียนบนปก:** `เขียนโดย ดร.เขมวันต์ บัวบาน`
- **ราคาปกติ:** `๕๐๐ บาท`
- **ราคาโปรโมชัน:** `๓๙๙ บาท`
- ราคาปกติให้วางบริเวณปกหลังใกล้ ISBN/Barcode
- ราคาโปรโมชันให้ทำเป็น Badge/Sticker Layer แยกจาก Artwork หลัก เพื่อถอดหรือเปลี่ยนได้
- ข้อความเสนอใช้บน Badge: `ราคาพิเศษ ๓๙๙ บาท จากราคาปกติ ๕๐๐ บาท`
- ต้องกำหนดระยะเวลาและเงื่อนไขโปรโมชันก่อนเผยแพร่สื่อขาย ห้ามแสดงเป็นส่วนลดถาวรโดยไม่มีเงื่อนไข
- Preview v0.3 แสดงข้อความตามคำสั่งแล้ว แต่ฉบับจริงต้องวางชื่อและราคาด้วย Live Text เพื่อป้องกันความคลาดเคลื่อนจากภาพสร้าง

ห้ามใช้ข้อความรับประกันผลลัพธ์ทางรายได้ เพราะผลจริงขึ้นกับคุณภาพผลิตภัณฑ์ ตลาด การขาย และการลงมือทำของผู้เรียน

## ข้อกำหนด Artwork ฉบับโรงพิมพ์

- ขนาด Trim เลือกหลังยืนยัน A5 หรือ B5
- Bleed อย่างน้อย 3 มม. รอบด้าน หรือตามข้อกำหนดโรงพิมพ์
- ภาพ Raster ที่ขนาดใช้งานจริง 300 PPI
- โลโก้และไอคอนใช้ Vector เมื่อมี
- ใช้ CMYK และ ICC Profile ตามโรงพิมพ์
- เก็บข้อความเป็น Live Text ใน AI/INDD/PSD และฝัง/แนบฟอนต์ที่มีสิทธิ์
- แยก Front Cover, Spine และ Back Cover ตามความหนาสันจริงหลังล็อกจำนวนหน้าและชนิดกระดาษ
- ส่ง PDF/X ตามมาตรฐานที่โรงพิมพ์ระบุ พร้อม Preflight Report

## สิทธิ์และการอนุมัติ

- ต้องยืนยันสิทธิ์ใช้ตราหน่วยงาน ชื่อหน่วยงาน ภาพบุคคล ภาพประกอบ ไอคอน และฟอนต์
- ต้องตรวจการสะกดชื่อผู้เขียน ผู้จัดทำ และหน่วยงานจากต้นฉบับที่อนุมัติ
- ห้ามส่งโรงพิมพ์จนกว่าจะมี Approval Record จากเจ้าของสิทธิ์หรือผู้มีอำนาจ

## Cover Approval Checklist

- [ ] ชื่อหนังสือตรงกับ Metadata
- [ ] ขอบเขตเครื่องมือตรงกับสารบัญเล่ม 1 ล่าสุด
- [ ] ไม่มีคำรับประกันรายได้หรือผลลัพธ์เกินจริง
- [ ] โลโก้และภาพบุคคลมีสิทธิ์ใช้งาน
- [ ] ขนาดปก Bleed Safe Area และ Spine ถูกต้อง
- [ ] ภาพ 300 PPI ที่ขนาดใช้งานจริง
- [ ] CMYK/ICC Profile ตรงข้อกำหนดโรงพิมพ์
- [ ] ตรวจคำไทยและชื่อเฉพาะแล้ว
- [ ] PDF/X ผ่าน Preflight
- [ ] บรรณาธิการใหญ่อนุมัติ Final Proof
- [ ] ชื่อผู้เขียนสะกดว่า “ดร.เขมวันต์ บัวบาน” ทุกตำแหน่ง
- [ ] ราคาปกติ ๕๐๐ บาทตรง Metadata และ Barcode Record
- [ ] Badge ราคา ๓๙๙ บาทเป็นเลเยอร์แยกและมีเงื่อนไขโปรโมชัน
