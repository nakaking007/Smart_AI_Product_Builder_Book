## จุดแข็งที่ให้รักษา

- โทน Navy Blue, Cyan และ Gold สอดคล้องกับ Series Art Direction
- ชื่อหลักเด่นและอ่านลำดับสายตาได้ดี
- มีภาพผู้เรียนลงมือใช้คอมพิวเตอร์และแท็บเล็ต
- ใช้ไอคอนอธิบายผลลัพธ์ เหมาะกับหนังสือ How To
- มีตำแหน่งเลขเล่ม ชื่อ Series และหน่วยงานชัดเจน

## ข้อความที่ต้องแก้ก่อนพิมพ์

| ตำแหน่ง | ข้อความปัจจุบัน | ข้อความเสนอใช้ |
| --- | --- | --- |
| แถบบน | คู่มือสร้าง AI Product ด้วยปัญญาประดิษฐ์ | คู่มือเรียนการใช้ AI ฉบับจับมือทำ |
| ชื่อรอง | จากผู้ใช้ AI สู่ผู้สร้าง AI Product | คงไว้ได้ |
| แถบชื่อเล่มล่าง | เล่มที่ 1: AI Foundation & Prompt Engineering | เล่มที่ 1: BASIC — Gem, Custom GPT, NotebookLM & Single Web App |
| กลุ่มเครื่องมือ | มี Google Apps Script เป็นแกน | เปลี่ยนแกนเป็น Single Web App; GAS เป็นตัวอย่างต่อยอดเท่านั้น |
| เหรียญทอง | “สร้างรายได้จริง” | “สร้างผลงานได้จริง • พร้อมต่อยอด” |
| คำโปรยระบบอัตโนมัติ | Google Apps Script และ Coding | Single Web App ด้วย HTML, CSS และ JavaScript |

ห้ามใช้ข้อความรับประกันผลลัพธ์ทางรายได้ เพราะผลจริงขึ้นกับคุณภาพผลิตภัณฑ์ ตลาด การขาย และการลงมือทำของผู้เรียน

## ข้อกำหนด Artwork ฉบับโรงพิมพ์

- ขนาด Trim เลือกหลังยืนยัน A5 หรือ B5
- Bleed อย่างน้อย 3 มม. รอบด้าน หรือตามข้อกำหนดโรงพิมพ์
- ภาพ Raster ที่ขนาดใช้งานจริง 300 PPI
- โลโก้และไอคอนใช้ Vector เมื่อมี
- ใช้ CMYK และ ICC Profile ตามโรงพิมพ์
- เก็บข้อความเป็น Live Text ใน AI/INDD/PSD และฝัง/แนบฟอนต์ที่มีสิทธิ์
- แยก Front Cover, Spine และ Back Cover ตามความหนาสันจริงหลังล็อกจำนวนหน้าและชนิดกระดาษ
- ส่ง PDF/X ตามมาตรฐานที่โรงพิมพ์ระบุ พร้อม Preflight Report

## สิทธิ์และการอนุมัติ

- ต้องยืนยันสิทธิ์ใช้ตราหน่วยงาน ชื่อหน่วยงาน ภาพบุคคล ภาพประกอบ ไอคอน และฟอนต์
- ต้องตรวจการสะกดชื่อผู้เขียน ผู้จัดทำ และหน่วยงานจากต้นฉบับที่อนุมัติ
- ห้ามส่งโรงพิมพ์จนกว่าจะมี Approval Record จากเจ้าของสิทธิ์หรือผู้มีอำนาจ

## Cover Approval Checklist

- [ ] ชื่อหนังสือตรงกับ Metadata
- [ ] ขอบเขตเครื่องมือตรงกับสารบัญเล่ม 1 ล่าสุด
- [ ] ไม่มีคำรับประกันรายได้หรือผลลัพธ์เกินจริง
- [ ] โลโก้และภาพบุคคลมีสิทธิ์ใช้งาน
- [ ] ขนาดปก Bleed Safe Area และ Spine ถูกต้อง
- [ ] ภาพ 300 PPI ที่ขนาดใช้งานจริง
- [ ] CMYK/ICC Profile ตรงข้อกำหนดโรงพิมพ์
- [ ] ตรวจคำไทยและชื่อเฉพาะแล้ว
- [ ] PDF/X ผ่าน Preflight
- [ ] บรรณาธิการใหญ่อนุมัติ Final Proof
